# Project Tech Hub

## Overview
Project Tech Hub is a full-stack web application designed to provide users with a comprehensive platform for learning and community engagement in technology. The project includes various pages that cover different aspects of the platform, including courses, community interactions, and personal learning experiences.

## Project Structure
The project consists of the following key components:

- **HTML Files**: Each page of the website is represented by an HTML file, which contains the structure and content of the page.
  - `about.html`
  - `community.html`
  - `contact.html`
  - `courses.html`
  - `home.html`
  - `index.html`
  - `projects.html`
  - `services.html`
  - `SignIn.html`
  - `SignUp.html`
  - `skills.html`
  - `teams.html`
  - `TechHub.html`
  - `whatILearning.html`

- **CSS Files**: Each HTML file is linked to a corresponding CSS file that contains styles specific to that page, ensuring a cohesive design throughout the website.
  - `about.css`
  - `community.css`
  - `contact.css`
  - `courses.css`
  - `home.css`
  - `index.css`
  - `projects.css`
  - `services.css`
  - `SignIn.css`
  - `SignUp.css`
  - `skills.css`
  - `teams.css`
  - `TechHub.css`
  - `whatILearning.css`

- **Assets**: The project includes folders for icons and logos that are used throughout the website.

## Setup Instructions
1. **Clone the Repository**: 
   ```
   git clone <repository-url>
   ```

2. **Navigate to the Project Directory**:
   ```
   cd Project-Tech-Hub-HTMLCSS
   ```

3. **Open the Project**: Open the project in your preferred code editor.

4. **Run the Project**: Open the `index.html` file in a web browser to view the website.

## Features
- User-friendly interface with easy navigation.
- Responsive design for optimal viewing on various devices.
- Comprehensive sections covering different aspects of technology learning and community engagement.
